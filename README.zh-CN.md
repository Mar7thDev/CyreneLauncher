# 🚀 Cyrene Launcher

[English](README.md) | **简体中文**

一个轻量、现代的游戏启动器 —— 让启动、更新与自定义游戏变得简单高效。

---

## ✨ 功能特性

- 🔄 启动时自动更新 March7thHoney 及代理工具
- 🎮 以正确的参数与环境启动游戏
- 🌐 通过语言工具切换游戏内语言（英文、日文、中文、韩文）
- 📦 使用 Hdiffz 工具（基于 HDiffPatch）应用游戏补丁

---

## 🧑‍💻 关于开发者

你好！我们是 **Mar7thDev**，一支热衷于打造简单而强大工具的开发团队。
Cyrene Launcher 使用以下技术构建：

- ⚙️ **Go + Wails** 实现前后端一体化

我们的目标是创造快速、高效、用起来令人愉悦的工具。

## 📦 安装

你可以选择：

- ✅ **便携版**（无需安装）
- 🛠️ **MSI 安装包**（完整集成）

---

## 🛠️ 常用开发命令（Wails v3）

```bash
# 以开发模式启动应用（前端热重载）
wails3 dev

# 构建应用（生产二进制）
wails3 build

# 打包应用（NSIS）
wails3 package
```
